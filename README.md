## 网站部署

### 免费部署方案（推荐）
1. **GitHub Pages**：免费托管，支持自定义域名
2. **Netlify**：免费CDN，自动部署
3. **Vercel**：快速部署，性能优秀

### 专业部署方案
1. **购买域名**：如 yanglawyer.com
2. **专业托管**：阿里云、腾讯云等
3. **HTTPS证书**：确保网站安全

## 使用方法

### 1. 本地预览
直接在浏览器中打开 `index.html` 文件即可预览网站。

### 2. 内容更新
- **文本内容**：直接编辑 `index.html` 文件
- **样式调整**：修改 `css/style.css` 文件
- **功能调整**：修改 `js/main.js` 文件

### 3. 图片替换
1. 将您的照片放入 `images/` 目录
2. 在 `index.html` 中更新图片路径
3. 确保图片大小合适，建议使用压缩工具优化
